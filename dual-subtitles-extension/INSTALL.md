## Installation Steps (5 minutes)

1. **Download the extension folder**
   - You have: `dual-subtitles-extension` folder
   - Keep all files together

2. **Open Chrome**
   - Type: `chrome://extensions/` in address bar
   - Press Enter

3. **Enable Developer Mode**
   - Look at top right corner
   - Toggle "Developer mode" ON (switch turns blue)

4. **Load the Extension**
   - Click "Load unpacked" button (appears after enabling Developer mode)
   - Browse to and select the `dual-subtitles-extension` folder
   - Click "Select Folder"

## First Use

### Option A: Start with FREE translation (No setup!)

1. Go to Netflix or YouTube
2. Play a video
3. Turn on English subtitles (CC button)
4. Sinhala subtitles appear at top automatically! üéâ

### Option B: Use Google Translate (Better quality)

1. Get API Key:
   - Visit: https://console.cloud.google.com/
   - Create new project
   - Enable "Cloud Translation API"
   - Create API Key (Credentials menu)
   - Copy the key

2. Configure Extension:
   - Click extension icon in Chrome
   - Select "Google Translate (Better Quality)"
   - Paste API key
   - Click "Save Settings"

3. Enjoy better translations!
